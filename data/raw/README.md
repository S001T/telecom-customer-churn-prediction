# Telecom Customer Churn Prediction

## Business Problem
Telecom companies lose customers every month.
The goal of this project is to predict which customers are likely to churn
so the business can take proactive actions to retain them.

## Dataset
IBM Telco Customer Churn dataset.

Target variable:
- Churn (Yes / No)

## Project Structure
data/
- raw/ – original dataset  
- processed/ – cleaned data  

notebooks/
1_data_cleaning.ipynb  
2_eda.ipynb  
3_feature_engineering.ipynb  
4_model.ipynb  

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Expected Outcome
- Churn prediction model
- ROC-AUC evaluation
- Business recommendations
